# StudyM 관리자 대시보드

StudyM 서비스의 관리자용 웹 대시보드입니다.

## 기술 스택

- React + TypeScript
- Vite (번들러)
- ESLint (코드 품질 관리)

## 개발 환경 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 주요 기능

- 상담 신청 관리
- 결제 내역 관리
- 회원 관리

## 관련 링크

- 메인 랜딩 페이지: `../src/`
- 백엔드 API: `study-manager-mvp/backend-django/`
